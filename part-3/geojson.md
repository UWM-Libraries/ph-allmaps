# Draw GeoJSON on a IIIF image

The georeference metadata produced by Allmaps Editor can be used to convert geospatial coordinates to pixel coordinates to draw GeoJSON on the original unwarped map.

## Paris stuff from Stanford
https://geodiscovery.uwm.edu/catalog?f%5Bpcdm_memberOf_sm%5D%5B%5D=stanford-zp399nk6394

This example uses a flood extent dataset derived from Philippe Buache’s 1744 map of the Seine to visualize the 1740 Paris flood. The data are then overlaid onto an 1821 map of Paris from the AGSL collection to compare how the same event aligns across different historical representations of the city.

    * https://collections.lib.uwm.edu/digital/collection/agdm/id/1550/ (AGSL Map of Paris, 1821)
    * see part-3/Crues_1740.geojson
    * see https://geodiscovery.uwm.edu/catalog/stanford-kt749jj7858
    * https://gallica.bnf.fr/view3if/ga/ark:/12148/btv1b8593335z
